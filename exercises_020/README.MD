    - Configure proxyChains
    
      Add your TOR to route your traffic to *ProxyList* section
      
      ```
      [ProxyList]
      # add proxy here ...
      # meanwile
      # defaults set to "tor"
      socks4  127.0.0.1 9050
      #http 192.168.0.1  3128
      ```
